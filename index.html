    La interfaz de usuario o UI (User Interface)

    De forma sencilla, podemos definirla como el medio a través del cual el usuario interactúa con un dispositivo tecnológico. Esto abarca todos los puntos de contacto entre la persona y el equipo. Para comprenderlo mejor, debes estar consciente de que cada vez que abres el correo electrónico, realizas una transferencia electrónica, envías un emoticono o cualquier otra acción similar estas comunicándote con un software y sistema operativo. Básicamente, transmites o señalas lo que deseas hacer y este responde. .
